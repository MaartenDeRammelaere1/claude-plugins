# Maarten's Claude Code Plugins

A collection of Claude Code plugins by Maarten De Rammelaere.

## Plugins

| Plugin | Description |
|--------|-------------|
| [thanks](./plugins/thanks) | Thank your AI when it does a great job, or sigh when it doesn't. |

## Installation

Add this marketplace to Claude Code:

```
/plugin marketplace add MaartenDeRammelaere1/claude-plugins
```

Then install any plugin:

```
/plugin install thanks@maarten-plugins
```
